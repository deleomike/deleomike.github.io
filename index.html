1. MIPS SuperScalar Processor Simulation

The Superscalar Processor Simulation was a research project that I did for my Computer Architecture
course. Essentially, it tested the idea that one could make a simulation of a processor that has several
inputs of instructions and can be used to study algorithms used to schedule the instructions such that the
instructions are executed quickly and with no obstructions present.

I accomplished the simulation portion, however I did not implement the algorithm. The room for an
algorithm is there, and the simulation is built around the implementation of some algorithm.
